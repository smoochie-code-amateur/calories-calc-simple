# Калькулятор Калорій
Максимально простий калькулятор калорій, для подальшого ручного внесення або розуміння. Можна вносити продукти та рецепти.

### Де База Даних?
Усе зберігається у Google Sheets, оскільки проєкт малий та для особистого використання. Я періодично чистю її, якщо бачу, що додали щось не те.

### Синхронізація профілю
Дані зберігаються локально у браузері + шифруються. Для цього треба буде створити пароль, який слугуватиме сигналом, що дані можна розшифрувати для імпорту та шифрувати для експорту. Можна переносити файл бекапу з пристроя на пристрій. Також, можна повністю очистити даніза потреби.

### Профіль
Тут зберігається ваш вік, вага, зріст, ім'я, стать та норма калорій. Нору калорій можна вирахувати з дефіцитоп та профіцитом. Після кожного оновлення сторінки вам доведеться вказуватип пароль, аби зайти. 
Також є Щоденник, де ви можете заносити те, що їли. Внизу вам покаже просту інфографіку калорій та речовин, аби було приблизне уявлення як ви харчуєтесь.
